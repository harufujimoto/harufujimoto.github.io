**ソートの話**

ソートするときに元の配列における添字が知りたいとき、pair<Value,id>のペアを作っても良いが、こうするとちょっと簡単。

(This code is not checked.)
{% highlight cpp %}
template<class T>
vector<int> GetIdOrder(vector<T>& vec){
  int N = vec.size();
  vector<int> res(N);
  iota(res.begin(),res.end(),0);
  sort(vec.begin(),vec.end(),[&](int i,int j){
    return vec[i] < vec[j];
  });
  return res;
}
